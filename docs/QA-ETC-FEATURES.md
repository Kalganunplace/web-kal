# 칼가는곳 QA - 기타 기능 테스트 케이스

## 📋 개요

이 문서는 칼가는곳 서비스의 기타 기능에 대한 QA 테스트 케이스를 정리한 문서입니다.

- **테스트 범위**: TC0039 ~ TC0082 (총 44개)
- **카테고리**: 홈, 가격표, 가이드, 네비게이션, 알림, 이용내역, 내정보, 쿠폰, 주소 설정, 공지사항, 고객센터, 이용약관, 설정
- **통과율**: 11/44 (25%)
- **실패율**: 33/44 (75%)

---

## 🎯 주요 실패 항목 요약

### 🔴 Critical (긴급)
- **TC0061**: 쿠폰 사용해도 차감 안됨, 결제 적용 안됨
- **TC0065**: 휴대폰 번호 변경 시 인증 절차 없음

### 🟠 High (중요)
- **TC0048**: 전체 읽음 버튼 작동 안함
- **TC0053**: 연도 선택 드롭다운 클릭 안됨
- **TC0056**: 같은 날짜에 서비스 2회 이상 시 날짜 중복 스택
- **TC0063**: 기존 회원정보 확인 불가
- **TC0064**: 이름 입력 시 확인 버튼 활성화 안됨

### 🟡 Medium (개선 필요)
- **TC0040**: 가격 표시 형식 수정 필요 ([정상 4,000원] → 4,000원)
- **TC0041**: 칼 상태 및 종류 관련 주의사항 오타 검토
- **TC0045**: 서브베너 레이아웃 수정
- **TC0050**: 이용내역 뒤로가기 버튼 삭제
- **TC0057**: 내정보 등급/횟수 정보 삭제
- **TC0059**: 보유쿠폰 빈 상태 UI 개선
- **TC0060**: 쿠폰 자세히보기 UI 간소화
- **TC0066**: 회원 탈퇴 확인 바텀싯 추가
- **TC0070**: 공지사항 기본 날짜 수정
- **TC0074**: 고객센터 FAQ/연락처 섹션 삭제
- **TC0076**: 이용약관 리스트 방식 변경
- **TC0077**: 개별 이용약관 클릭 오류

## TC0040: 개별 칼 클릭

### 화면
[가격표]

### Action
개별 칼 클릭

### Expected Result
- 칼 세부 내용 설명 바텀싯이 올라옴
- 칼 예시 사진, 칼 연마 시장가, 칼가는곳 전용 가격을 볼 수 있음

### 통과 여부
❌ **FALSE**

### 코멘트
추후에 현재 버전에서 칼 연마 세부 내용 설명이 빠져있으며, 쿠폰사용 시 할인가를 보여주는 형식으로 되어있음. 현 단계에서는 시장가, 칼가는곳 할인가만 보여주는 형식으로 변경되었으면 함
